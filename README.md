## Geral

### Funções implementadas

Os usuários devem ser capazes de:

- Adicionar uma nota
- Editar uma nota
- Mudar a cor de uma nota
- Deletar uma nota
- Acessar as notas mesmo depois de fechar a aplicação (fica salva no local storage)
